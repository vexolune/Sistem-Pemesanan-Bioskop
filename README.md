## Installation  

1. **Clone the project:**  
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory:**  
   ```bash
   cd <project-folder>
   ```

3. **Install the required dependencies:**  
   ```bash
   python -m pip install -r requirements.txt
   ```

4. **Customize the code as needed.**

5. **Run the application:**  
   ```bash
   python main.py
   ```
